HOW TO USE:

noise.py:
            1. Use "chmod +x noise.py" to make the file executable.
            2. Run the program with "./noise.py filename" where filename is the name of the image to be processed.
            3. When the program finishes executing the resulting images will be saved in the current directory under the names
               smoothedmedian.png smoothedalpha.png and smoothedkuwa.png.
